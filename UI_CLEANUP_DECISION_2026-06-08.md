## 📋 Phase 2 — 「수정 모드」 모달 통합 (= 다음 PR)

### 사장님 의도:
```
1. ✏️ 클릭 → 「수정 모드」 모달
2. 「현재가」 버튼 클릭 시:
   - 「↻ 설정만 수정」 모드 = 미진입 단계만 = 현재가 + 트리거율 재계산 ⭐
   - 「🛑 종료 후 새로 시작」 모드 = 모든 단계 재계산 (= 옛 동작)
3. 미리보기 → 사장님 확인
4. 「↻ 설정만 수정」 클릭 → 적용
```

### 구현 계획:
1. `fillStartPrice('current')` 함수 = 「수정 모드」 감지 추가
2. `_refreshLiveCalc()` 함수 = 진입 단계 trigger 보존 + 미진입 단계만 재계산
3. 「수정 모드」 모달 = 진입 단계 표시 (= ✅ 진입 배지) + 미진입 단계 시각 구분
4. 사장님 신 단계 추가 가능 (= 7~10단계 자본 입력)
