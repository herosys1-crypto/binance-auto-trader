→ 2x/5x/10x 무관 동일 (이전 ROI 기준 / leverage 정책 폐기)

트레일링 익절: 피크 ≥ +5% 후 피크 대비 -5% 회귀 시 잔량 100% 청산 (자동, stage>=3 + TP3+ 후).

🚨 크라이시스 모드란? (손절과 다른 안전망 — 펼쳐서 확인)

손절 (SL) 과 완전히 다른 안전망입니다. 포지션 청산이 아니라 TP 임계를 자동 낮춰 회복 시 빠른 익절을 유도합니다.

정상 TP 임계 (default)

TP1/2/3/4 = 10/15/20/30%

크라이시스 시 TP 임계 (자동)

TP1/2/3/4 = 5/10/15/20%

발동 조건: 누적 max_loss_pct ≤ 임계 (default -50%) + 모든 단계 진입 완료 (또는 「💉 포지션 추가」 사용)

손절과의 관계 (둘은 독립적):

• -50% 도달 → 🚨 크라이시스 진입 (TP 임계 낮춤, 청산 X)

• -80% 도달 → 🛑 손절 발동 (강제 전량 청산, 크라이시스 무관)

예시 흐름: -50% → 크라이시스 (TP 5%/10% 등으로 빨리 익절 시도) → 회복 못 하고 -80% 도달 → 손절 청산

⚠️ 본 모달의 신규 strategy = 크라이시스 비활성 (사용자 결정 2026-05-14, -100 sentinel).
→ 신규는 손절(SL)만 작동. 깊은 손실 시 회복 익절 모드 안 들어감.

📌 기존 strategy 는 옛 설정값으로 작동 중:
• 옛 default = -50% → 자동 크라이시스 진입 가능
• 예: #16 LABUSDT 같은 옛 strategy 가 현재 크라이시스 활성 = NULL/-50% 임계 도달
